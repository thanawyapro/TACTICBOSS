# Tactic Boss AI V130.2 — نسخة GitHub → Netlify الجاهزة

هذه هي نسخة الإطلاق الجاهزة بدون `npm install` أو Build على Netlify.

## ترتيب الملفات المطلوب في جذر GitHub

```text
dist/
netlify/
netlify.toml
READ_ME_FIRST_V130_2_PREBUILT_AR.md
```

ارفع العناصر الأربعة نفسها إلى جذر المستودع، ولا تضعها داخل مجلد خارجي.

## إعدادات Netlify

- Build command: موجود تلقائيًا داخل `netlify.toml` ويطبع رسالة فقط.
- Publish directory: `dist`
- Functions directory: `netlify/functions`

بعد استبدال النسخة القديمة:

1. اعمل Commit وPush إلى GitHub.
2. في Netlify افتح **Deploys**.
3. اختر **Trigger deploy → Clear cache and deploy site**.
4. بعد النشر افتح `/version.json?check=1302`.

الإصدار المتوقع:

```text
v130.2-web-production-final-launch-authentic-tactics-unified-ui
```
