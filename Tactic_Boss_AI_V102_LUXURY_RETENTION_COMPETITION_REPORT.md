# Tactic Boss AI V102 — Luxury UX + Board Collapse + Share Card Rebuild + Retention + Competition

## هدف الإصدار
تحويل التجربة من واجهة مبهرجة ومزدحمة إلى تجربة Premium أسهل وأهدأ، مع معالجة زحمة السبورة، إصلاح نصوص شاشة التحميل، وإعادة بناء كارت المشاركة العربي، وإضافة أساسيات Retention ومنافسة داخل التطبيق.

## التعديلات الرئيسية
- تبسيط الواجهة الرئيسية إلى Daily Command Center + 3 أفعال أساسية + أدوات ثانوية.
- تقليل الاعتماد على gradients القوية في كل كارت، وجعل التصميم أقرب إلى Premium Tactical Console.
- إضافة مركز اليوم: Streak، تحدي اليوم، ميتا اليوم.
- إضافة شاشة Coach League / دوري المدربين كبداية منافسة داخل التطبيق بدون SQL جديد.
- إضافة تحدي الخصم اليومي + ترتيب أسبوعي محلي + Badges تكتيكية.
- إصلاح شاشة التحميل لتتغير حسب السياق:
  - Build My Tactic
  - Counter Rival
  - Live Rescue
- جعل السبورة التنفيذية مغلقة افتراضيًا مع زر إظهار/إخفاء.
- نقل قراءة AI للسبورة إلى زر اختياري داخل كارت السبورة.
- جعل الخصم مخفيًا افتراضيًا داخل السبورة لتقليل الزحمة.
- تقليل حجم لاعبي السبورة في compact mode وإخفاء الخصم من board state حتى لو كان محفوظًا عند إغلاقه.
- إعادة بناء Share Card Canvas بمنطق أوضح للعربي والإنجليزي:
  - دعم RTL
  - منع خلط عربي/إنجليزي داخل السطر الأساسي
  - استخدام wrap text
  - تقليل النصوص الطويلة
  - استخدام اختصارات عالمية للمراكز داخل الملعب
  - تحميل الخطوط قبل الرسم عبر document.fonts.ready
- تحديث version/cache/manifest إلى V102.
