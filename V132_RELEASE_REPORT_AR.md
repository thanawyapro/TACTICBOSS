# Tactic Boss V132.0 — تقرير الإصدار والاعتماد

تاريخ الإصدار: 2026-06-18

## أهم ما تم تنفيذه

- إصلاح ربط نتيجة "ابنِ خطتك" بمدخلات المستخدم الظاهرة بدل القيم الافتراضية القديمة.
- فصل تقييم جودة الخطة عن نسبة الثقة في اكتمال وصحة البيانات.
- إضافة تحليل تفصيلي للهجوم والدفاع والتحولات والتوازن وملاءمة التشكيل والتعليمات.
- استكمال إعدادات PES الأساسية للهجوم والدفاع وتقسيمها بوضوح.
- دعم فروق PES 2019 وPES 2020 وPES 2021، ومنع Anchoring في PES 2019.
- فرض حد تعليمتين هجوميتين وتعليمتين دفاعيتين في PES.
- دعم تعيين اللاعب لتعليمات Defensive وCounter Target وAnchoring.
- دعم الرقابة اللصيقة على مركز من تشكيل الخصم مع إظهار المراقِب والهدف.
- إضافة الوضع السريع والوضع الاحترافي.
- إضافة خطوات واضحة لبناء الخطة وملخص حي للمدخلات.
- إضافة التحذيرات الفورية وكشف التعارضات مع إصلاح مقترح قابل للمعاينة.
- إضافة ثلاثة بدائل: آمن، متوازن، هجومي.
- إضافة خطط A وB وC وسيناريوهات التقدم والتأخر.
- إضافة ملف خصم مبسط وتأثيره على التحليل.
- إضافة سؤال المدرب عن المخاطرة الدفاعية.
- إضافة قائمة مراجعة نهائية قبل إنشاء النتيجة.
- إضافة الحفظ التلقائي لوضع البناء وبيانات الخطة.
- إصلاح ملف CSS المفقود الذي كان يمنع الـBuild.
- تحديث رقم الإصدار إلى 132.0.0 وملف version.json.
